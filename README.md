# Education-and-Population
This project uses Numpy and Pandas in Python by utilizing data cleaning, exploratory data analysis, and linear regression to answer the question: Are global years of education and number of out of school children correlated? This project submission was originally used as a final symposium project for Women in Computing and Data Science at UC Berkeley.

The following data set is used: https://www.kaggle.com/datasets/imtkaggleteam/global-education/data
